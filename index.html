CanvasPlot

CanvasPlot is a JavaScript plot library that is able to interactively visualize large data sets by using the element.

Usage

To create a new plot...

var plot1 = new CanvasDataPlot(d3.select("#maincontainer"), [1000, 900], {
    xAxisLabel: "IQ",
    yAxisLabel: "Test Score",
    markerLineWidth: 3,
    markerRadius: 5
});
