CanvasPlot

CanvasPlot is a JavaScript plot library that is able to interactively visualize large data sets by using the <canvas> element.

Usage

A plot object for a scatter plot can be created in an empty <div> with id plotcontainer via:

var scatterPlot = new CanvasDataPlot(d3.select("#plotcontainer"), [600, 350], {
    xAxisLabel: "IQ",
    yAxisLabel: "Test Score",
    markerLineWidth: 3,
    markerRadius: 5
});

To add data to scatterPlot, you need a data set consisting of an array of x/y pairs. The pairs have to ordered ascendingly by x value.

var data = [[-1,5], [0.5,6], [5,-2.5], [6,1], [10,9], [20,-55]];
scatterPlot.addDataSet("dataSet1", "Test Results", data, "steelblue");
